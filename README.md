# MealsApp
<h1>A Meals App for Android &amp; iOS (React Native)</h1>

<h3>Features:</h3>
<ul>
<li>React-Redux for state mangement</li>
<li>React Navigation 4 (StackNavigation, BottomTabnavigation, DrawerNavigation)</li>
</ul>

To build and run the project:<br>
`npm install`<br>
`expo start`
